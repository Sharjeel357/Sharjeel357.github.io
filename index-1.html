AI-Powered CSV Financial Chatbot

A natural-language chatbot for real-estate portfolio analysis over two CSV datasets (holdings & trades). A rule engine and Groq-powered classifier turn a plain-English question into a structured query plan, which a Pandas compute engine executes — the LLM never does the math, it only understands the question and writes the answer.

Built deliberately without LangChain or a vector database — the data is already structured, so a lean rule-based pipeline keeps it fast, explainable, and easy to debug.
